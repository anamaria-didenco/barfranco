## Colour
| Name        | Hex       | Use                                   |
|-------------|-----------|---------------------------------------|
| Franco Red  | `#BE1622` | Hero — headlines, marks, energy       |
| Charcoal    | `#1D1D1B` | Grounding — body, footers, contrast   |
| Cream       | `#F4EEE1` | Warmth — section grounds, cards       |
| Paper       | `#FCFBF7` | Canvas — default background           |

Rough balance: ~60% paper/cream · ~25% charcoal · ~15% red. Red is a spice, not the plate.

## Type
- **VTC Marsha Bold** — display, set UPPERCASE for headlines, marks, section titles.
- **Affairs (Regular + Italic)** — body serif. Use the italic for eyebrows, captions and warmth.

## The three marks
- **Wordmark** = the name / the class. Always the anchor.
- **Dachshund** = the charm / the wink. A playful accent, never the lead.
- **Negroni** = the ritual / the occasion. Aperitivo-Italian icon.

## Voice
Warm, direct, a little witty — never corporate. "Come say ciao," not "Contact us."
